# AtGlance-Mannequins
AtGlance Mannequins is a modern, responsive mannequin e-commerce and catalog website designed to showcase premium mannequin collections. The platform features product browsing, wishlist management, shopping cart functionality, WhatsApp-based ordering, and a seamless user experience across desktop, tablet, and mobile devices.



# ATGLANCE MANNEQUINS

**Tagline:** Display That Speaks

## About The Project

AtGlance Mannequins is a modern e-commerce and product showcase website designed for mannequin manufacturers, retailers, and wholesale buyers. The platform provides a premium shopping experience with a clean user interface, responsive design, product catalog management, wishlist functionality, shopping cart integration, and WhatsApp-based enquiry and order management.

The website is built to help businesses showcase mannequin collections professionally while making it easy for customers to browse products, save favorites, request quotations, and place orders.
